## Крестики‑нолики (игрок против компьютера)

Небольшая уютная игра на Python (Tkinter): вы играете **X**, компьютер играет **O**.

### Запуск

Установите зависимости и запустите:

```bash
python3 -m pip install -U requests
python3 main.py
```

### Telegram‑уведомления (опционально)

Если задать переменные окружения, игра отправит сообщение в Telegram‑бот:

- при победе: `Победа! Промокод выдан: [код]`
- при проигрыше: `Проигрыш`

Переменные:

- `TELEGRAM_BOT_TOKEN` — токен бота
- `TELEGRAM_CHAT_ID` — id чата/пользователя, куда отправлять сообщения

Пример (macOS / zsh):

```bash
export TELEGRAM_BOT_TOKEN="123456:ABCDEF..."
export TELEGRAM_CHAT_ID="123456789"
python3 main.py
```

Если переменные не заданы — отправка в Telegram **просто отключена**, игра работает как обычно.
